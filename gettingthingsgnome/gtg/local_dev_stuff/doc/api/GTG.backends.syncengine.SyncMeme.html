G.b.s.SyncMeme(object) : class documentation

Part of GTG.backends.syncengine View In Hierarchy

A SyncMeme is the object storing the data needed to keep track of the state of two objects synchronization. This basic version, that can be expanded as needed by the code using the SyncEngine, just stores the modified date and time of the last synchronization for both objects (local and remote)
