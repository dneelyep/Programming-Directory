def reset_cursor(self):
Returns the cursor to the tag that was selected prior to any right click action. Should be used whenever we're done working with any tag through a right click menu action.
def set_target_cursor(self):
Selects the last tag to be right clicked.

We need this because the context menu will deactivate (and in turn, call reset_cursor()) before, for example, the color picker dialog begins. Should be used at the beginning of any tag editing function to remind the user which tag they're working with.
