G.b.b.Backend(GenericBackend) : class documentation

Part of GTG.backends.backend_localfile View In Hierarchy

Localfile backend, which stores your tasks in a XML file in the standard XDG_DATA_DIR/gtg folder (the path is configurable). An instance of this class is used as the default backend for GTG. This backend loads all the tasks stored in the localfile after it's enabled, and from that point on just writes the changes to the file: it does not listen for eventual file changes
