G.b.s.SyncMemes(TwoKeyDict) : class documentation

Part of GTG.backends.syncengine View In Hierarchy

A TwoKeyDict, with just the names changed to be better understandable. The meaning of these names is explained in the SyncEngine class description. It's used to store a set of SyncMeme objects, each one keeping storing all the data needed to keep track of a single relationship.
