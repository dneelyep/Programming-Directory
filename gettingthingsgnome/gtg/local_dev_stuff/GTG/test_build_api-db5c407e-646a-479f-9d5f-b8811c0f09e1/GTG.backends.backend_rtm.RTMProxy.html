G.b.b.RTMProxy(object) : class documentation

Part of GTG.backends.backend_rtm View In Hierarchy

The purpose of this class is producing an updated list of RTMTasks. To do that, it handles:

  • authentication to RTM
  • keeping the list fresh
  • downloading the list
