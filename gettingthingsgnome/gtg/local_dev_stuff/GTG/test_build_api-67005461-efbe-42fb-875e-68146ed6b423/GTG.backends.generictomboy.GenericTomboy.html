def _tomboy_split_title_and_text(self, content):
Tomboy does not have a "getTitle" and "getText" functions to get the title and the text of a note separately. Instead, it has a getContent function, that returns both of them. This function splits up the output of getContent into a title string and a text string.
Parameterscontenta string, the result of a getContent call
