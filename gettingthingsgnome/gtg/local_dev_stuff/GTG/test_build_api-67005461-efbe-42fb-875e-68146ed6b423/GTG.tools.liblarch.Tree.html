Part of GTG.tools.liblarch View In Hierarchy

A thin wrapper to MainTree that adds filtering capabilities. It also provides a few methods to operate complex operation on the MainTree (e.g, move_node)

def get_viewtree(self, name=None, refresh=True):
This returns a viewtree. If name is given and a view of that name already exists, that existing view is returned. Else, a view with that name is created. If name is None, the view is not remembered. If refresh is False, the view is not initialized (meaning that it will probably not reflect the Tree. This is useful as an optimization if you plan to apply filter.
